As with most free and open-source software, a good way for any user to contribute is to simply use it and report any possible errors and feedback. The code for LeanHRPT is hosted on github, so the best way to file error reports and share feedback is using the following links:

> LeanHRPT-Decode GitHub
> LeanHRPT-Demod GitHub

If you manage to receive any particularly noteworthy satellite imagery using LeanHRPT, you can submit it to be featured in the gallery. To do so, you can use this submission form.

You can also support the author of LeanHRPT personally via Ko-Fi, if you are feeling extra generous.
